# Skills

The functional programming skill that previously lived here now has its own
repository so coding agents can install it directly:

**→ [functional-programming-composition/fp](https://github.com/functional-programming-composition/fp)**

[![skills.sh](https://skills.sh/b/functional-programming-composition/fp)](https://skills.sh/functional-programming-composition/fp)

```bash
npx skills@latest add functional-programming-composition/fp
```

That repository is the canonical source for the skill. It distills the working
contract taught across this curriculum — the error model (`Maybe` / `Either` /
`Validation`), the Functor → Applicative → Monad → Monoid → Traversable ladder,
branching by data shape, folds and trampolines in place of loops, the arity
policy, and the laws — into a form an agent can follow directly.

The lectures in this repository remain the long-form teaching material behind it.
